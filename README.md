# Laravel Blog Fundamentals

## 🎯 Tujuan
Project ini dibuat untuk mempraktikkan konsep dasar Laravel meliputi:
- Authentication & Authorization
- CRUD Operations
- Eloquent Relationships
- Blade Templating
- Migration & Seeding

## 🛠️ Tech Stack
- Laravel 10.x
- PHP 8.x
- MySQL
- Bootstrap/Tailwind CSS

## ⚙️ Fitur
- [ ] User Registration/Login
- [ ] Create/Read/Update/Delete Posts
- [ ] Categories & Tags
- [ ] Comments System
- [ ] Admin Dashboard
